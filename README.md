## Dotfiles 🚀

You will need to have **stow** installed to manage these dotfiles. You can install it using your package manager.

### Configuration

1. Clone the repository:

   ```bash
   git clone https://github.com/alevidals/dotfiles
   ```

2. Navigate to the cloned directory:

   ```bash
    cd dotfiles
   ```

3. Execute setup.sh script to setup the dotfiles:

   ```bash
   ./setup.sh
   ```
